sprite image

sprite image는 규모가 있는 중견기업 및 대형 쇼핑몰등에서 많은 아이콘 이미지(100px이하)를 효율적으로 관리하기 위해서 제작하는 이미지이다.

sprite image는 img태그가 아닌 css에서 background-image로 연결해서 사용한다.

날씨:비 검색
장바구니 좋아요 로고 지도 메뉴 검색
  1. 1. 아이콘의 의미를 HTML태그에 작성한다. ex) 검색, 바로가기, 홈, 위로, 아래로 등
  2. 2. 위 html를 css 선택자로 만들어서 background-image로 sprite 이미지로 연결한다.
  3. 3.sprite이미지의 개별위치,크기를 체크해서 css 입력한다.
  4. 4. (1)에서 작성한 의미있는 글자는 text-indent로 숨긴다.
